对象是Js的基础，Js中一共有6种主要类型

  • string
  • number
  • boolean
  • null
  • undefined
  • object

常见的一种错误：JS中万物接对象，显然是错误的。

Js中有许多特殊的对象子类型，可以称为复杂基本类型。

函数是对象的一个子类型，JS中函数是一等公民，本质上和普通的对象一样。

数组也是对象的一个类型，具备一些额外的行为。

  • String
  • Number
  • Boolean
  • Object
  • Function
  • Array
  • Date
  • RegExp
  • Error

在Js中，上面的内置对象实际上只是一些内置函数，可以当做构造函数来使用，从而可以构造出一个对应子类型的新对象。

举例来说

原始值strPrimitive并不是一个对象，因此判断instanceof String时返回false，但是如果要在这个字面量值上执行一些操作，比如获取字符长度等，JS会自动将其转换为String对象

但并不是所有的内置函数都有字面量和构造形式，有些只有一个，有些不管是字面形式还是构造形式都是对象

  • null和undefined只有字面形式
  • Date只有构造形式
  • Object，Array，Function，RegExp字面和构造形式都是对象。
  • Error很少显示创建，抛出异常时会自动创建，也可以使用new的方式，但通常用不上
